# 📚 StudyFlow

StudyFlow is a modern **study planner and productivity web application** designed to help students manage their time, tasks, and focus effectively.

It combines task management, study timers, analytics, and productivity tools in one simple interface to help students stay organized and motivated.

---

## 🚀 Features

* 📅 Study planner and task management
* ⏱️ Focus timer (Pomodoro style studying)
* 📊 Study analytics and progress tracking
* 🎵 Music player for focus sessions
* 🤖 AI assistant section
* 👥 Study groups feature
* 📆 Calendar-based planning system

---

## 🛠️ Built With

* **HTML5**
* **CSS3**
* **JavaScript**

These technologies make the app lightweight, fast, and easy to run directly in the browser.

---

## 💻 How to Run the Project

1. Download or clone this repository.
2. Open the main file:

```
index.html
```

3. Run it in any modern web browser.

No installation or server setup is required.
